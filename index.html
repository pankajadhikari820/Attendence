Smart Attendance System

Automated QR-based attendance management system with manual check-in and admin tracking.

About Project

This is a fully automated attendance system where users can register, generate a unique QR code, and mark attendance either via QR scanning or manual ID entry. Admins can view all attendance records, track statistics through graphs, and download Excel reports.

Project Pages & Features

Register Page

Users can register by entering their name and email. A unique registration ID and QR code are generated automatically. The QR code can be downloaded for future attendance marking.

Go to Register

Check-In Page

Users can mark attendance using their QR code or manual registration ID. Admins can access the check-in page with secure login. Multiple check-ins per day are allowed, and timestamps are recorded for accuracy.

Go to Check-In

Admin Login

Admins log in to access the dashboard. From there, they can view registered users, attendance records, daily and monthly graphs, attendance percentages, and export Excel reports for analysis.

Go to Admin Login
